初步了解WAMP服务器环境

WAMP是什么？

  • Windows
  • Apache
  • MySQL/ MariaDB
  • PHP /Python

WAMP的安装

手动安装方式

  • 先安装Apache，安装PHP组件并修改配置
  • 安装MySQL服务器环境
  • 安装MySQL的PHP扩展

利用集成环境

  • 集成环境是指将Apache、PHP、MySQL等服务器软件集成为一个单独的安装包，并包含默认配置
  • 常用的集成环境：WampServer、Xampp、Uniserver、EasyPHP等等。

安装WampServer

初始条件

  • 安装相应版本VC++可再发行组件包
  • 端口不能冲突：80（HTTP)、443(HTTPS)，3306(MYSQL)
    • HTTP可被嗅探
    • 冲突可能：其它Web服务器，如IIS
    • 了解冲突的方法：netstat -a

软件获取

  • 官方下载（注意先查看本机版本并下载相应的程序）
  • 执行安装（注意默认浏览器、编辑工具的选择）
  • 启动完成后
    • 绿色：所有服务已经启动
    • 橙色：至少一个服务未启动
    • 红色：所有服务未启动

    • localhost
  • Web根目录

所有的服务器环境默认都有一个WEB根目录
Wampser:WWW
Xampp: htdocs

访问规则：

WEB根目录：
本地：C:\wamp64\www
WEB: http://localhost/

_intro目录：
本地：C:\wamp64\www_intro
WEB: http://localhost/_intro

wordpress目录：
本地：C:\wamp64\www\wordpress
WEB: http://localhost/wordpress

local:C:\wamp64\www\wordpress-4.9.4-zh_CN\wordpress
web: http://localhost/wordpress-4.9.4-zh_CN/wordpress

语言包的在线安装
-site administration/language/language pack(s)

语言包的离线安装
-http://moodle.org下载语言包
-解压后放入moodledata\lang

个人默认语言设置
-Preferences\Useraccount\Preferred language

为不同类型的设备指定默认主题
-外观/主题选择器(默认自带的3个主题：Boost、Clean、More)

插件获取与安装方法1
-http://moodle.org上插件目录中获取zip文件
-在“网站管理/插件/安装插件”上传zip文件并安装

插件获取与安装方法2
-在“网站管理/插件/安装插件”
-“从Moodle插件目录安装插件”

建立概念：
Moodle中版块、活动类型、主题等等，全是插件

实践任务：
-访问http://gzs.ilester.net，下载往届学生的插件使用汇报，利用插件名称下载安装符合你的版本的安装包，并试用。

修改网站安全策略
-“网站管理/安全/网站安全设置”
-修改密码规则（或者禁用密码规则）

创建用户
-网站管理/用户/账户/添加用户
-依据网站的安全设置输入用户信息（默认添加的用户没有任何权限）

设置用户角色(系统管理员)
-网站管理/用户/权限/网站管理员
-添加或免除需要设置管理员的用户

设置用户角色(课程创建者)
-网站管理/用户/权限/系统自带角色
-添加或免除需要设置课程管理员的用户

批量导入用户
-创建一个用户信息表（.csv），字段:firstname, lastname, email, username, password
-“网站管理/用户/账户/上传用户”（注意CSV间隔符号，以及字符编码）

用上传用户实现批量选课
-创建CSV文件，基本字段：firstname, lastname, email, username, password
-选课字段：course1, type1, group1,course2, type2, group2,

courseN：课程简称
typeN：1,2,3
groupN:在课程中创建的小组名

导航设置（首页设置）
-网站管理/首页设置
-设置登录前后首页显示的信息区块

导航条的设置
-外观/主题设置
-导航菜单中添加菜单项（格式：菜单名|超链接 ，一行一条， 用-表示层级）

开启移动终端访问
-前提：Moodle与终端的物理上是连接的
-“网站管理/移动应用程序/移动设备设置”，开启设置

移动APP从哪里获取
-官方Moodle Mobile可以从应用市场下载，iOS、Android
-Windows 10系统：也可以下载UWP格式的客户端
